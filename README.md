# 🧺 Laundriku
## UTS Pemrograman Mobile

Laundriku adalah aplikasi mobile berbasis Android yang dirancang untuk memudahkan pelanggan dalam melakukan pemesanan jasa laundry secara praktis, cepat, dan efisien. Aplikasi ini memberikan kemudahan mulai dari pemilihan paket laundry hingga melihat status pesanan.

---

## 📱 Deskripsi Aplikasi

Laundriku merupakan solusi digital untuk membantu proses layanan laundry. Dengan aplikasi ini, pelanggan dapat melakukan pemesanan tanpa harus datang langsung ke tempat laundry.

Aplikasi dikembangkan sebagai proyek **Ujian Tengah Semester (UTS) Mata Kuliah Pemrograman Mobile**.

---

## 🎯 Tujuan Aplikasi

- Mempermudah pelanggan melakukan pemesanan laundry.
- Mengurangi proses pencatatan secara manual.
- Memberikan informasi status pesanan secara cepat.
- Meningkatkan kualitas pelayanan laundry.

---

## ✨ Fitur Aplikasi

- Login Pengguna
- Dashboard
- Pilih Paket Laundry
- Form Pemesanan Laundry
- Riwayat Pesanan
- Profil Pengguna
- Logout

---
